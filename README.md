# Visual Information Presentation – Final Project

## Overview
This project was developed as a final assignment for the course  
**Visual Information Presentation & Cognition**.

The project presents an interactive Tableau visualization that enables exploration
and comparison of survey data from **2022 and 2025** regarding internet usage patterns
among different sub-groups within the Ultra-Orthodox (Haredi) community.

## Research Goal
- Compare internet usage patterns across sub-groups (Hasidic, Lithuanian, Sephardic, Other)
- Analyze demographic distributions (age, gender)
- Identify trends and changes over time
- Support non-technical decision-makers through guided visual storytelling

## Data
- Survey data from **2022** and **2025**
- Data preparation included:
  - Translation to English
  - Standardization of fields across years
  - Feature engineering (age groups, usage frequency, access methods)
  - Union of datasets in Tableau

## Visualization Design
- Multiple dashboards connected through interactive filters and actions
- Storyboard structure following:
  **Overview → Zoom & Filter → Details on Demand**
- Chart types include:
  - 100% stacked bar charts
  - Grouped bar charts
  - Donut charts
  - Line charts for trends over time
